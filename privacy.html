Narratio — Политика конфиденциальности

Дата вступления в силу: 06.09.2025

Это приложение (Narratio) предоставляет тестовые функции клиентского интерфейса и обращается к бэкенду по адресу https://narratio-backend-5asu.onrender.com. Мы стремимся обрабатывать данные прозрачно и минимально.

1. Какие данные обрабатываются

  • Технические данные запроса: IP-адрес, заголовки браузера/устройства, время запроса — могут логироваться хостингом (Render, GitHub Pages) для обеспечения безопасности и стабильности.
  • Данные, введённые пользователем: только то, что вы добровольно отправляете через интерфейс. На текущем этапе приложение отправляет тестовые запросы к эндпоинтам /api/health и /api/test без персональных данных.
  • Файлы cookie/хранилище: фронтенд может использовать локальное хранилище/кэш браузера для улучшения работы интерфейса. Персональные данные здесь не сохраняются.

2. Цели обработки

  • Обеспечение работы приложения и ответов API.
  • Диагностика ошибок и защита от злоупотреблений.
  • Улучшение стабильности и производительности.

3. Правовые основания

Обработка основана на законных интересах по обеспечению функционирования сервиса и вашей инициативе при использовании функций приложения. Политика ориентирована на принципы GDPR (минимизация, прозрачность, безопасность).

4. Передача и хранение

  • Фронтенд размещён на GitHub Pages, бэкенд — на Render (ЕС-регион). Логи хостинга могут храниться ограниченное время для безопасности.
  • Мы не продаём и не передаём персональные данные третьим лицам в маркетинговых целях.

5. Сроки хранения

Технические логи хранятся хостинг-провайдерами в разумные сроки, необходимые для безопасности и отладки. Пользовательские данные в тестовой версии не сохраняются целенаправленно.

6. Права пользователей

Вы вправе обращаться за разъяснениями и удалением данных, если они были предоставлены. Свяжитесь с нами по адресу ниже.
